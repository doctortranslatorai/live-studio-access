# LIVELOOP — Rede social apenas para lives

Mockup front-end em HTML, CSS e JavaScript.

## Páginas

- `index.html` — feed principal de lives
- `explore.html` — descoberta por categoria
- `live.html` — sala de live com chat
- `create.html` — criar live em modo mockup
- `profile.html` — perfil do criador
- `messages.html` — mensagens privadas
- `style.css` — design global
- `app.js` — dados simulados, filtros, chat e localStorage

## Como publicar no GitHub Pages

1. Enviar todos os ficheiros para a raiz do repositório.
2. Ir a Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Save.
